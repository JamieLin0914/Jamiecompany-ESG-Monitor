Jamiecompany-ESG-Monitor (EcoPay)

這是一個專為大型活動場館設計的 智慧化 ESG 能源監控平台。
透過即時點位模擬與碳排計算，協助管理員即時掌握用電狀況並評估碳費成本。
目標是鎖定中小型企業，幫助他們在碳稅這方面能夠節省一些，甚至成為智慧樓宇，達到永續發展。

核心亮點
實際操作如下(假設)
- **即時電力監控**：針對 1F (A/B/C 區) 與 4F (D/E/F 區) 進行多點位電力負載模擬。
- **碳排即時預估**：自動結合電力排放係數（0.424 kg CO2e/度）換算即時碳排。
- **ESG 財務決策**：預估碳費支出，並根據用電熱點提供節能控制建議。
- **動態視覺化**：整合 Plotly 儀表板，即時呈現各設備（冷氣、燈光、插座）耗電佔比。

## 🛠️ 技術棧
**語言**: Python
**框架**: Streamlit
**數據**: Pandas, NumPy
**視覺化**: Plotly (GoBar, Pie)

## 🚀 快速啟動
1. 安裝環境：`pip install -r requirements.txt`
2. 啟動平台：`streamlit run app.py`
